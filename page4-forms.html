What is a Form?

Forms collect user input like names, emails, and feedback. They use the <form> tag.

<form action="submit.php" method="post">   Name: <input type="text" name="name">   <input type="submit" value="Submit"> </form>
Exercise: Create a form that takes name, email, and a submit button.

Input Fields

Different types of input fields:

<input type="text" placeholder="Enter text"> <input type="password" placeholder="Enter password"> <input type="email" placeholder="Enter email"> <input type="number" placeholder="Enter number"> <input type="checkbox"> Agree <input type="radio" name="gender"> Male <input type="radio" name="gender"> Female
Exercise: Add a password and email field in your form.

Textarea

The <textarea> tag is used for multi-line text input.

<textarea rows="4" cols="30">Write your feedback here...</textarea>
Exercise: Add a textarea in your form for feedback.

Select & Datalist

<select> creates a dropdown. <datalist> provides auto-suggestions.

<select>   <option>HTML</option>   <option>CSS</option>   <option>JavaScript</option> </select> <input list="browsers"> <datalist id="browsers">   <option value="Chrome">   <option value="Firefox">   <option value="Edge"> </datalist>
Exercise: Add a dropdown for selecting a country.

Fieldset & Legend

These group related inputs in a form.

<fieldset>   <legend>User Info</legend>   Name: <input type="text">   Email: <input type="email"> </fieldset>
