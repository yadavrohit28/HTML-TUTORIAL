What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It structures text, links, images, and multimedia. CSS adds style, while JavaScript adds interactivity.

How HTML Works

HTML documents are plain-text files saved with a .html extension. Browsers read them, interpret the tags, and render formatted content.

'Hello World' Example

<!DOCTYPE html>
<html>
<head>
  <title>My First Page</title>
</head>
<body>
  <p>Hello World</p>
</body>
</html>
