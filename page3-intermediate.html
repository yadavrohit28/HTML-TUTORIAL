HTML Formatting Tags

HTML provides several tags to format text:

  • <b> – Bold text
  • <i> – Italic text
  • <u> – Underlined text
  • <mark> – Highlighted text
  • <small> – Smaller text
<p>This is <b>bold</b>, <i>italic</i>, and <u>underlined</u></p>
Exercise: Create a paragraph where one word is bold, another is italic, and another is highlighted.

Div and Span

<div> groups block-level elements, while <span> is for inline styling.

<div style="background:gray;padding:10px;">   This is inside a div. </div> <p>This is <span style="color:orange">highlighted</span> text.</p>
Exercise: Make a div with a background color and put two paragraphs inside it.

Class and ID

Classes and IDs help style and identify elements:

<style> .highlight { color: orange; } #unique { font-weight: bold; } </style> <p class="highlight">This uses a class.</p> <p id="unique">This uses an ID.</p>
Exercise: Add two paragraphs with the same class and style them. Add one unique element with an ID.

File Paths

File paths tell the browser where to find files (like images).

  • Absolute Path: Full URL (e.g. https://example.com/images/pic.jpg)
  • Relative Path: Location relative to current file (e.g. images/pic.jpg)
<img src="images/photo.jpg" alt="My Photo">
Exercise: Create a folder "images", add a picture, and display it using a relative path.
